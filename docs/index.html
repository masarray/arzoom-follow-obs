Five-minute setup

Made for OBS beginners.

No Python, no scripts panel, no external tracking application, and no manual crop-filter automation.

Download and install

Click Download to receive the latest Windows installer directly. Close OBS, run it, then restart OBS.

Add it to Display Capture

Open Display Capture → Filters → Effect Filters → + and select ArZoom - Smart Mouse Zoom.

Open Hotkeys from the filter

Click Open OBS Hotkeys Settings, then assign a shortcut to ArZoom — Toggle Zoom & Mouse Follow.

Use the recommended defaults

Start with 2.00× zoom, Smart follow, Smooth movement, and a 28% stable safe zone.

Press once, then press again

The first press zooms and follows. The second press recenters and returns to the normal full-screen view.

FAQ

Before you install

Does ArZoom work with Window Capture or Game Capture?

The current preview focuses on Display Capture. Window Capture and Game Capture are outside the v0.x MVP so the first release can remain predictable and lightweight.

Will it reveal black areas near the edge of the monitor?

No by design. The viewport center is constrained by the current zoom level, so the visible area remains within the source texture even at corners.

Why does the frame stop before the mouse reaches the center near an edge?

That is the correct edge-safe behavior. The viewport cannot keep the cursor centered and remain inside the monitor at the same time, so ArZoom prioritizes valid video output.

Is the current release production-stable?

v0.1.4 is a public Windows preview. The core design is implemented, but wider OBS 32.x, GPU, mixed-DPI, 4K, and long-duration streaming validation is still underway before v1.0.

Does ArZoom collect telemetry?

No. ArZoom has no analytics service and does not transmit screen or cursor data.
